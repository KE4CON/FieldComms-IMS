🔵 YAAC Setup Guide

1. Install YAAC
Download from ka2ddo.org/ka2ddo/YAAC.html
Requires Java 8+: sudo apt install default-jre

2. Configure YAAC web server port
In YAAC: File → Configure → Web Server tab
Set port to 8082 (avoids conflict with Graywolf on 8080)
Enable: Enable REST API ✓   Enable WebSocket ✓

3. Connect your TNC / radio
YAAC → Configure → Ports → Add Port
• Serial TNC: select COM port, baud 9600
• KISS TNC: uses TCP, e.g. Dire Wolf on port 8001
• APRS-IS: internet gateway (server: noam.aprs2.net:14580)
• AGW: for AGWpe / sound card TNC

4. Run as a service (optional)
Create /etc/systemd/system/yaac.service:

[Unit]
Description=YAAC APRS Client
After=network.target

[Service]
Type=simple
User=fieldcomms
WorkingDirectory=/opt/yaac
ExecStart=/usr/bin/java -jar /opt/yaac/YAAC.jar -nogui
Restart=always

[Install]
WantedBy=multi-user.target
Then: sudo systemctl enable --now yaac

5. Verify connection
This page will show 🔵 YAAC as connected when it responds on port 8082.
Click ⟳ Poll on the Sources tab after YAAC is running.
