Tunnel Control

Tunnel controls require physical access to the gateway Pi. These buttons only work when this page is open in the Chromium browser on the gateway Pi itself — not from operator laptops or phones on EMCOMM-NET. This is by design. A valid callsign login is also required.

To control the tunnel: sit at the gateway Pi keyboard → open Chromium → go to http://localhost:9001 → log in with your callsign → use the controls.
